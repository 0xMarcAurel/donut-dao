# Donut DAO

The Donut DAO is a decentralized organization created to support and scale the Donut ecosystem beyond the boundaries of Reddit. Built on the principles of community collaboration and governance, the Donut DAO rewards content creators and contributors who drive the project forward.

> Our mission is to expand DONUT's use cases, promote decentralization through SocialFi, and build an ecosystem that bridges Web3 with meaningful real-world impact.

---

## About this

This repository contains the official website of the Donut DAO.

- Written in HTML, CSS, and JavaScript
- Originally static, intended to be deployed as an informational site
- Open-source for transparency, but active development is limited to Donut DAO contributors

---

## Getting started

To run the site locally:

```bash
git clone https://github.com/0xMarcAurel/donut-dao.git
cd donut-dao
```

Open `index.html` directly in your browser, or use Live Server in VS Code
